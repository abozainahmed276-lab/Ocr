# محرك الملازم (Malaasem Engine)

## وصف المشروع
أداة رقمية وتفاعلية مصممة لأتمتة وتسهيل إنشاء أوراق العمل، المذكرات، والاختبارات التفاعلية لمادة الرياضيات.

## أهداف المشروع
* تطبيق منهجية "السهل الممتنع" في تبسيط القوانين الهندسية والرياضية.
* توفير الوقت والجهد في إعداد مراجعات ليلة الامتحان للطلاب.
* إنشاء بيئة اختبار مرنة تتيح عرض الأسئلة وحلها وتجربتها بشكل مستقل.

## التقنيات المستخدمة
* HTML, CSS, JavaScript
